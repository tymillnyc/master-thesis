МГУ имени М.В.Ломоносова, Факультет вычислительной математики и кибернетики, Магистерская диссертация 2025 / Lomonosov Moscow State University, Faculty of Computational Mathematics and Cybernetics, Master’s Thesis, 2025

**Тема:** Методы аугментации данных на примере задачи классификации текстов на русском языке / Data Augmentation Methods: A Case Study of Russian Text Classification

**Аннотация:**  В данной работе рассматривается влияние аугментации данных на качество моделей обработки естественного языка. Реализована библиотека с методами аугментации текстовых данных для русского языка и проведено исследование их эффективности в задаче классификации текстов.

Реализованная в рамках работы библиотека с методами аугментации: [репозиторий](https://github.com/tymillnyc/augmenta-ru).
